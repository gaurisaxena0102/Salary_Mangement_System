# Salary Management System

## 📌 Overview

This is a simple **Salary Management System** developed using **Python and MySQL**.
It allows users to manage employees, departments, and salary details through a terminal-based interface.

---


## 🛠️ Technologies Used

* Python
* MySQL
* MySQL Connector (Python library)
* Streamlit(for frontend)

---


## ▶️ How to Run the Project

1. Install Python

2. Install required libraries
  
3. Setup MySQL database

4. Run the SQL file:
- Schema.sql

5. Update database credentials in:
db_connection.py

6. Run the application: 

## 📊 Database Tables

* Department
* Employee
* Permanent Employee
* Intern
* Salary
