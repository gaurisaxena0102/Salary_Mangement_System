# Salary Management System

## 📌 Overview

This is a simple **Salary Management System** developed using **Python and MySQL**.
It allows users to manage employees, departments, and salary details through a terminal-based interface.

---


## 🛠️ Technologies Used

* Python
* MySQL
* MySQL Connector (Python library)

---


## ▶️ How to Run

1. Install Python and MySQL
2. Create database and run `schema.sql`
3. Install required packages:
   pip install mysql-connector-python
4. Run the program:
   python main.py

---

## 📊 Database Tables

* Department
* Employee
* Permanent Employee
* Intern
* Salary
